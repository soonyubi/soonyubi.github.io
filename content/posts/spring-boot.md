## spring boot ?

It is open source framework that allows for the configuration of an application of production scale with minimal setup.

It is incorporates servlet containers like Tomcat, Jetty, and Undertow, allowing the construction of a web server without separate configuration, and it can add Spring Boot Startups, which automate complex dependencies and configurations.
